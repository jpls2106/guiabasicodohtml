    • Client X Server

    (Voltar)

    Na comunicação entre Servidor(Server), Cliente(Client) e Usuário, o fluxo ocorre da seguinte maneira: O Servidor disponibiliza uma resposta para o Cliente, fornecendo a página solicitada ou arquivos para download.
    Ele interpreta as requisições recebidas do Cliente, que enviam informações como formulários e arquivos para o Servidor. O Cliente, por sua vez, exibe uma saída para o Usuário, mostrando a página renderizada abordada do Servidor em formato HTML. Além disso, o Cliente interpreta como inspirador do Usuário, que introduz dados diferentes no Cliente, os quais podem ser enviados eventualmente ao Servidor.

    O que é o HTML?

    (Voltar)

    A Linguagem de Marcação de Hipertexto (HTML) é amplamente utilizada na composição de páginas da internet e aplicativos online.
    O HTML permite a criação de hipertextos, que consistem em textos com referências a outros conteúdos. Trata-se de uma linguagem de marcação composta por uma série de marcações que indicam aos servidores da web a estrutura e o estilo de um documento.
    Embora o HTML não seja considerado uma linguagem de programação, uma vez que não possui recursos para criar funcionalidades dinâmicas, ele possibilita aos usuários criar e organizar seções, parágrafos e links utilizando elementos, tags e atributos.


    Como é uma estrutura básica de um HTML?

    (Voltar)

    A estrutura fundamental do HTML consiste nas tags < !DOCTYPE > , < head > e < body>.
    Essas três tags principais são essenciais para definir a estrutura básica de uma página web. Embora seja necessário utilizar outras tags para adicionar conteúdo à página, internamente, o HTML é construído e executado por meio dessas três tags principais.


    O que são TAGS?

    (Voltar)

    As TAGS do HTML têm a finalidade de criar diferentes tipos de elementos. Cada tag começa com a estrutura "< nome_da_tag >" e termina com "< /nome_da_tag >" (sem o ponto). Portanto, é importante prestar atenção no nome da tag que estamos utilizando enquanto mantidos com HTML.
